# ZZABIS - AI 음성 타이핑 도우미

macOS/Windows용 AI 음성 타이핑 앱입니다. 핫키를 누르고 말하면 AI가 음성을 인식하여 텍스트로 변환하고, 선택한 말투 스타일로 자동 입력합니다.

## 주요 기능

- **음성 인식**: OpenAI Whisper API를 통한 정확한 한국어 음성 인식
- **스타일 변환**: GPT-4o-mini를 활용한 10가지 말투 스타일 변환
- **자동 타이핑**: 음성을 텍스트로 변환하여 현재 커서 위치에 자동 입력
- **맞춤법 수정**: AI 기반 자동 맞춤법 교정
- **Push-to-Talk**: 핫키를 누르고 있는 동안만 녹음

## 스타일 모드 (10가지)

| 모드 | 설명 | 예시 |
|------|------|------|
| 그대로 | 맞춤법만 수정 | 원본 유지 |
| 공적 | 격식체 존댓말 | ~습니다, ~합니다 |
| 정중 | 공손한 존댓말 | ~해요, ~세요 |
| 반말 | 친구끼리 반말 | ~야, ~어, ~지 |
| 귀엽게 | 귀여운 말투 | ~용, ~당, ~해용 |
| 애교 | 애교 섞인 말투 | ~잉, ~쪄, ~행 |
| 다정 | 따뜻한 말투 | ~해줄게, ~고 싶어 |
| 쿨하게 | 담담한 말투 | ~임, ㅇㅇ |
| 유머 | 재미있게 | 드립/재치 추가 |
| 비즈니스 | 전문가 말투 | ~드립니다 |

## 설치 방법

### 1. 저장소 클론

```bash
git clone https://github.com/your-repo/zzabis.git
cd zzabis
```

### 2. 가상환경 생성 및 활성화

```bash
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# 또는
venv\Scripts\activate  # Windows
```

### 3. 의존성 설치

```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
source venv/bin/activate
python main.py
```

## 초기 설정

### 1. OpenAI API 키 설정

앱 실행 시 API 키 입력 창이 나타납니다.
- https://platform.openai.com/api-keys 에서 발급
- 음성 인식(Whisper) + 스타일 변환(GPT-4o-mini) 모두 사용

### 2. 핫키 설정

설정 버튼(⚙)을 클릭하여 원하는 핫키를 설정합니다.
- **마우스**: 사이드 버튼, 휠 버튼
- **키보드**: 단일 키 또는 조합 (Ctrl+Alt+Z 등)

### 3. macOS 권한 설정

처음 실행 시 다음 권한이 필요합니다:

1. **마이크 권한**: 음성 인식을 위해 필요
   - 시스템 설정 → 개인 정보 보호 및 보안 → 마이크

2. **접근성 권한**: 키보드 입력 및 핫키 감지를 위해 필요
   - 시스템 설정 → 개인 정보 보호 및 보안 → 접근성
   - Terminal.app 또는 실행 중인 앱 추가

## 사용 방법

1. 앱 실행
2. 원하는 스타일 버튼 클릭 (상단에 10개 버튼)
3. 설정된 핫키를 **누른 상태**로 말하기
4. 핫키를 떼면 음성 인식 → 스타일 변환 → 자동 입력
5. 자동으로 Enter 키가 입력됨

### 특수 명령어

- **"엔터"**: Enter 키만 입력 (텍스트 입력 없이)

## 문제 해결

### 핫키가 작동하지 않아요
- 접근성 권한이 부여되었는지 확인하세요
- 다른 앱과 핫키가 충돌하지 않는지 확인하세요

### 음성이 인식되지 않아요
- 마이크 권한이 부여되었는지 확인하세요
- 설정에서 올바른 마이크 장치가 선택되었는지 확인하세요

### "초기화 중" 메시지가 계속 나와요
- 첫 실행 시 OpenAI API 연결에 시간이 걸릴 수 있습니다
- 인터넷 연결을 확인하세요

### API 오류가 발생해요
- API 키가 올바르게 입력되었는지 확인하세요
- OpenAI API 사용량 한도를 확인하세요
